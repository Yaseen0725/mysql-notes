# mysql-notes
What is SQL?
SQL (Structured Query Language) is a standard programming language used to interact with relational databases. It is used for querying, inserting, updating, deleting, and managing data stored in databases.

Why we use SQL?
Simplifies working with large datasets.
Enables easy and efficient data management.
SQL is universal and works with most database systems (like PostgreSQL, MySQL, SQL Server).

Data : Data refers to raw facts, figures, observations, or values that can be processed to generate meaningful information.

Database : A database is an organized collection of data that can be easily accessed, managed, and updated digitally.

DBMS (Database Management System): It is software used to create, store, retrieve, manage, and manipulate data in a database.
